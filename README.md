#微营销管理平台
<br/>
##报名插件部分
报名插件是使用在手机端的H5微站。页面分为：<br/>
1、课程详情页（index.html）；<br/>
2、报名填写页（signup.html）；<br/>
3、公司简介页（company.html）；<br/>
4、报名反馈页（seccess.html）；<br/>
<br/>
###涉及到的技术、框架及效果模块
1、手机端布局（手淘flexible）；<br/>
2、轮播（swiper）；<br/>
3、时间轮（mobiscroll）；<br/>
4、表单验证（正则//test）；<br/>
5、多行多出隐藏及展开（clamp）；<br/>
